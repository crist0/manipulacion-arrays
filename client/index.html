Curso de manipulación de Arrays en JavaScript

Clases

forEach

Hace un recorrido por cada elemento del array

Código ejemplo:

array.forEach(item => console.log(item));
Reto 01-for-each.html

Map

Recorrido y modifica cada elemento del array

Código ejemplo:

const nuevoArray = letters.map(item => item + '++');
02-map.html

Filter

Filtra en un nuevo array los elementos que cumplan con la condición

Código ejemplo:

const nuevoArray = letters.filter(item => item.total >= 100);

reduce

Reduce a un objeto o valor los elementos del array realizando la condición, ejemplo

Código ejemplo:

const total = numArray.reduce((suma, item) => suma + item, 0);
Reto 05-reduce.html

some

Retorna true si consigue que un elemento del array cumple con la condición

Código ejemplo:

const res = array.some(item => item % 2 === 0);

every

Retorna True si todos los elementos del array cumplen con la condición

Código ejemplo:

const total = numArray.every(item => item < 40);
Reto

find

Retorna el objeto o valor del elemento a buscar

Código ejemplo:

const res = array.find(i => i === 30);

find index

Retorna la posición del elemento a buscar en el array

Código ejemplo:

const res = array.findIndex(i => i === 30);
